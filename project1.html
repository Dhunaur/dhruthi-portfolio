Skincare Store — PHP & MySQL e-commerce

Project overview

A fully functional skincare e-commerce store built from scratch with PHP and MySQL. Users can browse products, manage a session-based cart, register an account, log in, and complete checkout — all without a single JavaScript framework. Built to understand how web apps work at the server level.

Project type Full Stack / PHP
Year 2025
My role Developer
Project link github.com/Dhunaur/skincare-website
  • Built a multi-page PHP store with session-based cart — add, update, and remove products persisted across pages without a database call on every click.
  • Implemented user authentication with registration, login, and logout flows, using MySQL for user records and PHP sessions for state.
  • Designed the full checkout pipeline — cart → order summary → place order → thank you — with MySQL-backed order storage and a fallback JSON product catalog for offline dev.
  • Wrote all frontend styles from scratch: sticky header, hero section, product grid, and a responsive cart page — no CSS framework, just clean vanilla CSS.
