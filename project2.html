Blooming Delights — React flower shop

Project overview

A full-stack flower shop built with React on the frontend and an Express + MongoDB backend. Users can browse a live product catalog, sign up, log in, and place orders with billing details and coupon support. The first project where I wired a React app to a real REST API I wrote myself.

Project type Full Stack / Web App
Year 2025
My role Designer & Developer
Project link github.com/Dhunaur/flower-website
  • Built a React SPA with React Router — Home, Products, Product Detail, Checkout, and Login pages — each with its own CSS module and clean component separation.
  • Wrote an Express.js backend with MongoDB (Mongoose) handling product listings, user signup/login with bcrypt password hashing, and order storage.
  • Designed the full checkout flow: product selection → billing form → coupon code validation → order placed and persisted to MongoDB.
  • Handled CORS, form validation, and error states on both ends — the biggest learning was understanding how the frontend and backend need to agree on data shapes.
